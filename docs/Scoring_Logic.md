# 📐 점수 산출 기준서 (Scoring Criteria)

반려견 산책로 위험도 분석 파이프라인에서 사용하는 세 가지 점수의 **산출 기준과 근거**를 정리한 문서입니다.

> 모든 점수는 0~100 범위로 정규화되며, 높을수록 해당 위험/특성이 강함을 의미합니다.  
> (cushion_score는 예외: 높을수록 보행감이 좋음)

---

## 1. Heat Risk — 지면 열 위험도

### 개요

여름철 지면 온도에 의한 반려견 발바닥 화상 위험을 수치화한 점수입니다.  
국립기상과학원 실측 기준 아스팔트(최대 약 55°C)와 흙길(약 32°C)의 온도 차이(20°C 이상)를 점수 설계의 근거로 삼았습니다.

### 산출 구조

```
heat_risk = 재질 베이스 점수 + 배수 등급 가산점
```

surface 데이터가 없을 경우, 토성(DEEPSOIL) + 배수(SOILDRA) 조합으로 대체 산출하며,  
토양 데이터도 없으면 highway 유형의 포장 확률로 최종 추정합니다.

### 재질 베이스 점수

| 재질 (surface 값) | 베이스 점수 | 근거 |
|-------------------|-------------|------|
| `asphalt`, `concrete`, `paved`, `concrete:plates`, `Braille_blocks`, `paving_stones` | 70 | 인공 포장재. 직사광선 흡수율 높아 지면 온도 급상승 |
| `cobblestone`, `stone`, `cobblestone:flattened`, `sett`, `gravel`, `rock` | 45 | 석재류. 열 보유력 높으나 인공 포장보다 낮음 |
| `urethane`, `tartan`, `rubber`, `compacted` | 30 | 탄성 포장 또는 다진 흙. 상대적으로 온도 상승 낮음 |

### 배수 등급 가산점 (heat_drain_val, Max +25)

| 배수 등급 (SOILDRA) | 가산점 | 근거 |
|---------------------|--------|------|
| `매우양호` | +25 | 수분 유지 못해 건조·과열 위험 높음 |
| `양호` | +20 | |
| `약간양호`, `약간 양호` | +15 | |
| `약간불량`, `약간 불량` | +12 | |
| `매우 불량` | +10 | |
| `불량` | +8 | 수분 잔류로 증발 냉각 효과 있음 |
| 데이터 없음 | +5 | 보수적 기본값 |

### 토성 보조 점수 (surface 없을 때 대체, Max +30)

| 토성 (DEEPSOIL) | 점수 | 근거 |
|-----------------|------|------|
| `식질` | 30 | 점토 입자. 열 보유력 최고 |
| `미사식양질`, `역질`, `사력질` | 25 | |
| `사질`, `식양질` | 20 | |
| `미사사양질` | 15 | |
| `사양질` | 10 | 모래 성분 높아 열 분산 빠름 |
| 기타/없음 | — | 배수 가산점으로만 산출 |

### highway 기본 추정값 (토양 데이터도 없을 때)

| highway | heat_risk | 근거 |
|---------|-----------|------|
| `residential` | 60 | 대부분 아스팔트·콘크리트 포장 |
| `footway` | 55 | 보도블록 포장 비율 높음 |
| `path` | 50 | 비포장 비율 상대적으로 높음 |
| `track` | 48 | 농로. 흙길 비율 높음 |
| 기타 | 52 | 중간값 |

> **설계 배경**: 초기에 surface 결측 시 기본값을 20점으로 고정했을 때, 전체 136,034행 중 75,107행(55%)이 동일 점수에 몰리는 분포 편향이 발생했습니다. highway 유형별 포장 확률 기반 추정으로 개선했습니다.

## 2. Roughness Score — 노면 거칠기 점수

### 개요

반려견 발바닥 상처 및 관절 염좌 위험을 수치화한 점수입니다.  
노면 거칠기와 경사도가 함께 관절 부하를 결정한다는 수의학 연구(Millard et al., 2010)를 근거로, 경사도 가중치를 추가로 적용합니다.

### 산출 구조

```
roughness_score = base_roughness × 경사도 가중치
```

base_roughness는 OSM smoothness 실측값을 최우선으로 적용하고,  
없으면 자갈함량 → 토성 → surface 재질 → highway 순으로 단계적 추론합니다.

### Level 0. OSM smoothness 실측값 (최우선)

| smoothness 값 | base_roughness | 설명 |
|---------------|----------------|------|
| `excellent` | 15 | 매우 평탄 |
| `good` | 30 | 양호 |
| `intermediate` | 50 | 보통 |
| `bad` | 80 | 불량 |
| `very_bad` | 100 | 매우 불량 |

### Level 1. 자갈 함량 (smoothness 없을 때)

| 자갈 함량 (SUR_STON) | base_roughness | 근거 |
|----------------------|----------------|------|
| `>35` | 80 | 자갈 다량. 발바닥 상처·미끄러짐 위험 높음 |
| `10-35` | 60 | 자갈 중등량 |
| `<10` | 30 | 자갈 소량. 상대적으로 안정적 |

### Level 2. 토성 (자갈 데이터도 없을 때)

| 토성 (DEEPSOIL) | base_roughness | 근거 |
|-----------------|----------------|------|
| `식질`, `식양질`, `미사식양질` | 50 | 점토질. 건조 시 균열, 우천 시 미끄러움 |
| `사질` | 40 | 모래질. 발목 빠짐 위험 |
| `사양질`, `미사사양질` | 20 | 상대적으로 안정적 |

### Level 3. surface 재질

| surface 값 | base_roughness | 근거 |
|------------|----------------|------|
| `rock`, `stone`, `cobblestone`, `gravel`, `unpaved` | 80 | 불균일 석재·비포장 |
| `Braille_blocks` | 65 | 점자블록 요철 구조 |
| `cobblestone:flattened` | 55 | 평탄화 자갈. 일반 자갈보다 낮음 |
| `earth`, `ground`, `sand`, `compacted` | 50 | 자연 비포장 |
| `concrete:plates` | 35 | 판형 콘크리트. 이음새 있음 |
| `paving_stones`, `paved` | 30 | 일반 보도블록 |
| `asphalt`, `concrete` | 20 | 균일한 포장. 거칠기 낮음 |
| `urethane`, `wood` | 15 | 탄성 포장. 가장 낮음 |

### Level 4. highway 기본값 (모든 데이터 없을 때)

| highway | base_roughness |
|---------|----------------|
| `steps` | 70 |
| `residential` | 40 |
| `footway`, `path`, `track` | 55 |
| 기타 | 45 |

### 경사도 가중치

| 조건 | 가중치 | 근거 |
|------|--------|------|
| avg_slope ≥ 15% | ×1.3 | 급경사. 후지 하중 급증 (Millard et al., 2010) |
| avg_slope ≥ 8% | ×1.1 | 완만한 경사. 관절 부하 증가 시작점 |
| avg_slope < 8% | ×1.0 | 가중치 없음 |

> 최종 점수는 100점 상한으로 클리핑합니다.

## 3. Cushion Score — 보행감(쿠션) 점수

### 개요

지면의 충격 흡수력을 수치화한 점수입니다. 높을수록 발바닥·관절에 가해지는 충격이 적습니다.  
토심이 깊고 모래 성분이 많을수록 쿠션이 좋고, 점토질이 많거나 인공 포장일수록 쿠션이 낮습니다.

### 산출 구조

```
cushion_score = 토심 베이스 점수 + 토성 보정값
```

인공 탄성 포장재가 있으면 100점으로 즉시 확정하고,  
surface 데이터가 없으면 highway 유형으로 최종 추정합니다.

### 토심 베이스 점수 (cushion_base)

| 유효 토심 (VLDSOILDEP) | 베이스 점수 | 근거 |
|------------------------|-------------|------|
| `100초과` | 90 | 깊은 토층. 충격 흡수력 최고 |
| `50-100` | 70 | |
| `20-50` | 40 | |
| `기타` (얕음) | 25 | 충격 흡수력 낮음 |

### 토성 보정값 (cushion_soil)

| 토성 (DEEPSOIL) | 보정값 | 근거 |
|-----------------|--------|------|
| `미사사양질`, `사양질` | +15 | 모래 입자. 탄력적이며 발바닥에 편안함 |
| `사질` | +10 | |
| `식양질`, `미사식양질` | -5 | 점토 성분 증가. 딱딱해짐 |
| `식질` | -15 | 점토질. 딱딱하고 충격 흡수 낮음 |
| 기타/없음 | 0 | 보정 없음 |

### surface 재질별 고정 점수

| surface 값 | cushion_score | 근거 |
|------------|---------------|------|
| `urethane`, `tartan`, `rubber`, `wood` | 100 | 인공 탄성 포장재. 충격 흡수 최상 |
| `unpaved`, `ground`, `dirt`, `earth`, `sand`, `compacted`, `grass` | 토심+토성 계산 | 자연 비포장. 토양 데이터로 정밀 계산 |
| `cobblestone:flattened` | 20 | 평탄화 자갈. 딱딱하나 완전 포장보단 높음 |
| `Braille_blocks` | 15 | 요철 구조. 충격 흡수 낮음 |
| `cobblestone`, `stone` | 12 | 딱딱하고 불균일 |
| `asphalt`, `concrete`, `concrete:plates`, `paved`, `paving_stones`, `sett` | 10 | 인공 포장. 충격 흡수 거의 없음 |

### highway 기본 추정값 (surface 없을 때)

| highway | cushion_score | 근거 |
|---------|---------------|------|
| `steps` | 5 | 계단. 충격 흡수 최저 |
| `residential` | 15 | 아스팔트·콘크리트 추정 |
| `footway`, `path` | 45 | 비포장 비율 상대적으로 높음 |
| `track` | 55 | 농로. 흙길 비율 높음 |
| 기타 | 25 | 중간값 |

> 최종 점수는 0~100 범위로 클리핑합니다.

## 4. 경사도 수치 변환 기준

VWorld SOILSLOPE는 범위 문자열로 제공되므로 구간 중앙값으로 변환합니다.

| SOILSLOPE 원본값 | avg_slope (변환값, %) |
|------------------|-----------------------|
| `0-2%` | 1.0 |
| `2-7%` | 4.5 |
| `7-15%` | 11.0 |
| `15-30%` | 22.5 |
| `30-60%` | 45.0 |
| `60-100%` | 80.0 |

### slope_type 분류 기준

| 조건 | slope_type | 근거 |
|------|------------|------|
| highway == `steps` | `계단` | 수치 경사도 의미 없음 |
| avg_slope ≤ 3.0% | `평지` | 국토교통부 보도 설치 기준 (평지 ≤ 3%) |
| avg_slope ≤ 8.0% | `완만` | |
| avg_slope > 8.0% | `급경사` | 수의학 연구상 후지 하중 급증 기점 (Millard et al., 2010) |
| 데이터 없음 | `정보없음` | |

### highway 유형별 경사도 기본값 (공간 조인 미매칭 시)

| highway | 기본 avg_slope (%) | 근거 |
|---------|--------------------|------|
| `steps` | NULL | 수치 경사도 의미 없음 |
| `track` | 7.0 | 농로. 일반적으로 완만한 경사 |
| `living_street`, `path` | 3.5 | 생활 도로 |
| `footway` | 2.5 | 보행 전용 도로 |
| `residential`, 기타 | 1.5 | 주거지 진입로. 대부분 평탄 |

## 5. 안전 등급 (final_safety_grade) 판정 기준

위험 요소를 우선 판정한 뒤, 긍정 등급 순으로 분류합니다.

| 우선순위 | 조건 | final_safety_grade |
|----------|------|--------------------|
| 1 | roughness_score ≥ 80 | `위험 (발바닥 부상 주의)` |
| 2 | heat_risk ≥ 80 | `주의 (여름철 화상 주의)` |
| 3 | surface ∈ {urethane, tartan, rubber} | `매우 안전 (탄성 포장)` |
| 4 | cushion_score ≥ 75 | `매우 안전 (푹신한 흙길)` |
| 5 | surface ∈ {asphalt, concrete, paving_stones, sett} | `평범 (아스팔트/보도블록)` |
| 6 | 해당 없음 | `보통 (일반 노면)` |

## 6. filter_attributes 태그 생성 기준

UI 필터링에 사용되는 태그 배열의 포함 조건입니다.

| 태그 | 포함 조건 |
|------|-----------|
| slope_type 값 (`평지` / `완만` / `급경사` / `계단`) | 항상 포함 |
| `계단있음` | highway == `steps` |
| `계단없음` | highway ≠ `steps` |
| `푹신함` | cushion_score ≥ 70 |
| `지면뜨거움` | heat_risk ≥ 70 |
| `거친길` | roughness_score ≥ 60 |
